## 1. 프로젝트 소개
매번 해외 기술 블로그나 IT 뉴스를 번역기 돌려가며 읽는 번거로움을 해결하기 위해 만든 '맞춤형 뉴스 웹 앱'입니다. 지정한 RSS 피드에서 글을 자동으로 긁어와 오픈AI의 `gpt-4o-mini` 모델을 통해 핵심만 3줄의 한국어로 깔끔하게 요약하여 카드 형태로 보여줍니다.

## 2. 주요 기능
- **RSS 자동 수집:** 주요 IT 매체(TechCrunch, The Verge 등)의 최신 피드 수집
- **AI 3줄 요약:** 긴 영문 본문을 토큰 제한에 걸리지 않게 전처리하여 한국어로 요약
- **카드 뉴스 UI:** Streamlit을 활용한 직관적이고 깔끔한 격자형 레이아웃 화면
